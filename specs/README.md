# Specs

Esta pasta reune notas iniciais e futuras especificacoes do Rinha das Lendas.

Os arquivos atuais sao deliberadamente curtos. Eles registram a direcao do
projeto sem antecipar entidades definitivas, autenticacao ou detalhes de
integracao.

## Arquivos iniciais

- `001-estrutura-inicial.md`: organizacao tecnica do repositorio.
- `002-mvp-rinha-das-lendas.md`: escopo inicial do produto para refinamento.

Quando o Spec Kit for inicializado com `make speckit-init`, use estes arquivos
como contexto para criar a constituicao, a spec detalhada e o plano tecnico.
