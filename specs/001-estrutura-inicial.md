# 001 - Estrutura inicial

## Objetivo

Preparar o repositorio para iniciar o desenvolvimento do Rinha das Lendas com
backend e frontend separados, ambiente reproduzivel e documentacao orientada
por specs.

## Decisoes iniciais

- Backend em .NET 10 com API, Application, Domain, Infrastructure e Tests.
- Frontend em Vue 3 com Vite, TypeScript, Vue Router, Pinia e Axios.
- PostgreSQL executado por Docker Compose.
- Desenvolvimento principal dentro de Dev Container.
- Spec Kit disponivel no container e inicializado sob demanda.

## Fora de escopo

- Regras de negocio.
- Entidades definitivas e migrations.
- Autenticacao.
- Integracoes com Discord, Riot API, OP.GG ou DeepLoL.
- Telas completas.
