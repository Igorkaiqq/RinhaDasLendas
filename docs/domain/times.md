# Times

Times representam equipes reutilizaveis para partidas, drafts e consultas futuras.

## Regras

- Um time ativo precisa ter nome e tag.
- Nome e tag nao podem duplicar outro time ativo.
- Um jogador nao pode aparecer duas vezes no mesmo time.
- Times ativos aceitam apenas jogadores ativos.
- A composicao principal tem no maximo cinco jogadores.
- O capitao deve fazer parte da composicao.
- Inativacao preserva o registro para historico.
